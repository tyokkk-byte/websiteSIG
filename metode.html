Metode Penelitian dan Pengumpulan Data

Penelitian ini menggunakan metode deskriptif kuantitatif dengan tujuan untuk menggambarkan kondisi luas panen, produksi, dan produktivitas kacang tanah di Kabupaten Semarang. Data diperoleh dari publikasi resmi Badan Pusat Statistik (BPS), sehingga akurat, valid, dan dapat dipertanggungjawabkan.

1. Metode Penelitian

Pendekatan penelitian yang digunakan adalah metode deskriptif. Data yang diperoleh dari BPS dianalisis dan disajikan dalam bentuk visualisasi peta digital agar kondisi pertanian per kecamatan dapat dipahami secara lebih jelas dan sistematis.

2. Metode Pengumpulan Data

Pengumpulan data dilakukan menggunakan metode dokumentasi, yaitu mengambil data luas panen, produksi, dan produktivitas kacang tanah dari dokumen resmi BPS Kabupaten Semarang, yaitu:

  • Tabel statistik pertanian kacang tanah per kecamatan
  • Publikasi pertanian tahunan
  • Data produktivitas komoditas tanaman pangan

3. Teknik Pengolahan Data

Setelah data diperoleh, pengolahan dilakukan melalui langkah-langkah berikut:

  • Rekapitulasi dan penyesuaian format data
  • Pembersihan data (data cleaning)
  • Konversi data ke format tabel untuk QGIS
  • Menggabungkan (join) data statistik ke peta batas kecamatan

4. Metode Pemetaan

Pemetaan dilakukan menggunakan QGIS dengan teknik peta tematik (choropleth map) berdasarkan nilai luas panen, produksi, atau produktivitas. Peta kemudian diekspor ke dalam format web map menggunakan plugin qgis2web.

5. Penyajian Data pada Website

Data yang telah diolah ditampilkan dalam website menggunakan HTML, CSS, dan JavaScript. Konten yang disajikan meliputi:

  • Peta interaktif panen kacang tanah
  • Tabel data resmi dari BPS
  • Informasi detail per kecamatan
